\section{Dependent Type Checking}

The motivation for weak-head normal forms in dependent type theory is
to strike a balance between simplicity and efficiency in the
implementation of the type checker.

$$
\infer[]
  {\Gamma \turn f~a : B[a]}
{
  \Gamma \turn f : \Pi~A~B
  &
  \Gamma \turn a : A
}
\qquad
\infer[]
  {\Gamma \turn a : B}
{
  \Gamma \turn a : A
  &
  A \conv B
}
$$
